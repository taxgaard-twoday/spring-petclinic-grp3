# Booking af tider hos dyrlæger

## Titel

Som forretningsansvarlig for PetClinic ønsker jeg, at mit system kan håndtere booking af tider hos dyrlæger, så ejere kun kan booke gyldige tider hos relevante specialister.

## Pitch

PetClinic understøtter registrering af besøg, men ikke planlægning af fremtidige aftaler. Klinikken ønsker et bookingsystem, hvor ejere kan finde ledige tider, booke en aftale, aflyse en aftale og kun vælge dyrlæger, der passer til dyrets behov.

## Forretningsmæssige acceptkriterier

- En ejer kan se ledige tider for en valgt dyrlæge.
- En ejer kan booke en gyldig tid.
- En ejer kan aflyse en tid, hvis reglerne tillader det.
- Systemet forhindrer dobbeltbookinger.
- Systemet forhindrer booking hos en dyrlæge med forkert speciale.
- Fejlbeskeder skal være forståelige nok til, at frontend eller API-bruger kan reagere på dem.
- Systemet kan demonstreres via API-kald og docker compose.
